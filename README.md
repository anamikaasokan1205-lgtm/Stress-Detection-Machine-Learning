# Stress-Detection-Machine-Learning

## Overview
This project uses machine learning to detect stress from text data (e.g., social media posts).  
The final model is deployed as a Streamlit application, allowing users to enter text and receive real-time predictions.

## Features
- Text preprocessing pipeline (tokenization, cleaning, vectorization)  
- Machine learning model for binary stress classification  
- Interactive Streamlit interface for user input and predictions  

## Tools
- Python  
- scikit-learn  
- Streamlit  

## Future Work  
- Explore deep learning approaches for improved accuracy  
- Add multilingual support and richer response options
